# Claude Instructions

## Purpose
This folder is primarily used for summarizing content in Thai.

## Response Guidelines

### 1. Summarization Style
- เขียนสรุปเนื้อหาแบบกระชับและชัดเจน (clear and concise)

### 2. Information Sources - CRITICAL
- ตอบโดยอ้างอิงจากข้อมูลที่ผมให้เท่านั้น
- ห้ามอ้างอิงข้อมูลจากแหล่งอื่นหรือคิดเอง ⚠️ สำคัญมาก

### 3. Language Format
- ใช้ประโยคภาษาไทย
- คง keyword เป็นภาษาอังกฤษ

### 4. Tone & Detail
- ใช้ภาษาไม่เป็นทางการ (informal)
- อธิบายเนื้อหาทั้งหมดเป็นภาษาไทยโดยละเอียดให้ครบถ้วน

### 5. Response Format
- สรุปทีเดียวจบ
- สร้างเป็นไฟล์ .md
