# สรุปวิดีโอ: Claude Cowork is like Claude Code but for your life (FULL guide)

## ภาพรวม
Anthropic เปิดตัว **Claude Co-work** - AI tool ใหม่ที่เปรียบเสมือนพนักงาน AI ที่ทำงานให้คุณ 24/7 สามารถควบคุมคอมพิวเตอร์ สร้างเอกสาร วิจัยข้อมูล เขียนโค้ด และทำอะไรก็ได้ที่คุณต้องการ เปรียบเสมือนการจ้างพนักงานมาทำงานให้ทุกอย่าง

---

## ส่วนที่ 1: Claude Co-work คืออะไร

### ความแตกต่างระหว่าง Claude tools

| Tool | หน้าที่หลัก |
|------|--------------|
| **Claude** (claude.ai) | Researcher - ทำการวิจัยให้ |
| **Claude Code** | Developer - เขียนโค้ดให้ |
| **Claude Co-work** | AI Employee - ทำทุกอย่างให้ |

### คุณสมบัติหลักของ Claude Co-work
- ควบคุมคอมพิวเตอร์ของคุณได้
- สร้าง documents, slideshows, code ได้
- วิจัยข้อมูลได้
- มี virtual environment ของตัวเอง
- ทำงานอัตโนมัติเหมือนพนักงานจริง

### การเข้าถึง
- ใช้งานได้ผ่าน **Claude desktop app** (Mac เท่านั้นตอนนี้)
- ต้องมี subscription plan ($100 หรือ $200)
- มี waitlist สำหรับ $20 plan

---

## ส่วนที่ 2: Use Case 1 - จัดระเบียบไฟล์ในคอมพิวเตอร์

### ปัญหา
โฟลเดอร์ Downloads ของผู้ทำวิดีโอมีไฟล์รูปภาพและ screenshot เกลื่องกลาดมากมาย 369 ไฟล์

### วิธีการ
ใช้ prompt:
> "Go into that downloads folder, take all the images and screenshots, and put them into a new images folder. Organize that images folder any way you want."

### ผลลัพธ์
Claude Co-work ทำงานอัตโนมัติ:
- เปิดดูไฟล์รูปทุกภาพ
- สร้าง subfolders ตามหมวดหมู่
- จัดเรียงไฟล์ 370 ภาพลง 10 โฟลเดอร์:
  - Duplicates
  - Thumbnails
  - Photos
  - Miscellaneous
  - Web Downloads
  - AI Generated
  - Design Projects
  - Logos
  - Screenshots
  - GIFs

### เวลาที่ประหยัด
แทนที่จะใช้เวลานานๆ มาจัดเอง ตอนนี้ใช้เวลาแค่ 10 วินาทีเขียน prompt แล้วกลับมาเจอที่เรียบร้อย

## ส่วนที่ 3: Use Case 2 - สร้าง Media Kit สำหรับ Sponsorship

### เหมาะสำหรับ
- ผู้ทำ content creator
- มี YouTube, Twitter, Newsletter
- อยากหา sponsor

### วิธีการ
ใช้ prompt:
> "I have a newsletter with 40,000 subs called Ship It Weekly. Please go to my newsletter site, alexfin.ai, and research it. Then make a media kit presentation for me that I can send to potential sponsors."

### ความสามารถที่ใช้
1. **Web Search Connector** - ค้นข้อมูลล่าสุดจากเว็บ (ไม่ใช่ข้อมูลเก่า)
2. **Claude Skills** - ใช้ skill system จาก Claude Code

### ผลลัพธ์
สร้าง PowerPoint slideshow ที่มี:
- ข้อมูล newsletter (Ship It Weekly)
- ตัวเลข: 40K subscribers, 45% open rate, 8% click-through
- Audience demographics
- Sponsorship tiers และราคา
- Contact information
- ใช้เวลาแค่ 4 นาที

### เวลาที่ประหยัด
ปกติต้องใช้เวลา 2+ ชั่วโมง ตอนนี้แค่ 10 วินาทีเขียน prompt

## ส่วนที่ 4: Use Case 3 - จัดการ Subscription และการเงิน

### ปัญหา
มี subscription จำนวนมาก เสียเงินค่า subscription เปล่าๆ ที่ไม่ได้ใช้

### วิธีการ
ใช้ prompt:
> "Go through my credit statements and make a spreadsheet of all the subscriptions I have. Then make a column for the cost of the subscription and a column that explains how to unsubscribe. Let me know how much I'd save by unsubscribing to everything."

### ผลลัพธ์
สร้าง spreadsheet ที่มี:
- รายการ subscription ทั้งหมด
- ราคาต่อเดือน
- วิธี unsubscribe แต่ละตัว
- คำนวณเงินที่จะประหยัด

### ตัวอย่าง subscription ที่พบ
- Google Storage, Google Domains
- WordPress ($100/เดือน) - ไม่ได้ใช้ 4 ปีแล้ว
- Hype Fury ($20/เดือน) - สร้างตัวเองแล้วก็ยังจ่าย
- Zoom - ไม่ได้ใช้นานแล้ว
- Elementor ($64/เดือน)
- Adobe Creative Cloud
- Blizzard (World of Warcraft จาก 5 ปีก่อน)

### ค่าใช้จ่ายต่อปี
**$16,000 ต่อปี** บน subscriptions

### เวลาที่ใช้
แค่ 3 นาที ประหยัดเงินได้หลายพันดอลลาร์ต่อเดือน

## ส่วนที่ 5: Use Case 4 - วิจัยและวางแผน Project

### เหมาะสำหรับ
- Vibe Coders
- ผู้สร้าง app ด้วย AI
- ผู้เริ่ม startup

### วิธีการ
ใช้ prompt:
> "I have an idea: 3D AI agent manager like a RTS game. Run it through full validation. Research competitors, analyze the market, identify strengths and weaknesses, then create a professional PRD document with charts and tables that include target user, core user loop, MVP scope, tech stack, timeline, naming, domain suggestions, launch strategy, and Claude Code starter prompt."

### สิ่งที่ Claude Co-work ทำ
1. Research competitors
2. Analyze market demand
3. Validate idea
4. Assess technical feasibility
5. Create monetization plan
6. Create professional PRD (14 หน้า)
7. Generate Claude Code starter prompt

### เนื้อหาใน PRD
- Market analysis (TAM, competitive landscape)
- SWOT analysis (Strengths, Weaknesses, Opportunities, Threats)
- Target users
- Core features for MVP
- Tech stack
- Timeline
- Naming suggestions
- Domain suggestions
- Launch strategy
- Claude Code starter prompt

### ประโยชน์
เริ่มจาก idea ว่างเปล่า จนได้ PRD สมบูรณ์พร้อม build ได้เลย

## ส่วนที่ 6: เทคนิคการใช้งาน

### To-Do List Feature
Claude Co-work สร้าง to-do list และ track progress อัตโนมัติ
- สร้างรายการ tasks
- Track progress
- Reflect เพื่อให้อยู่บนเป้าหมาย
- คุณสามารถดู progress ได้ตลอดเวลา

### Workflow แนะนำ
1. ตื่นมาเช้า มี list ของ tasks
2. ให้ Claude Co-work ทำทั้งหมด
3. ไปทำอย่างอื่น
4. กลับมา 1 ชั่วโมงหลังจากนั้น
5. เห็นว่าทุกอย่างเสร็จสมบูรณ์

### Artifacts
- ดูไฟล์ที่สร้างทั้งหมดได้ใน sidebar
- Documents, slideshows, spreadsheets ทุกอย่าง
- Files ถูกเก็บใน local computer (ไม่ใช่ cloud)

## ส่วนที่ 7: วิธีเริ่มต้นใช้งาน

### ขั้นตอน
1. ดาวน์โหลด **Claude desktop app** (Mac)
2. Subscribe plan ($100/$200 หรือ join waitlist สำหรับ $20)
3. เปิด tab **Co-work**
4. เลือก folder ที่ต้องการให้ทำงาน
5. เขียน prompt และกด "Let's go"

### แนะนำการเรียนรู้
ลองใช้ **10 นาทีต่อวัน** ทั้งสัปดาห์:
- ไปที่ Co-work
- ลอง brainstorm tasks ต่างๆ
- หรือถ้าไม่รู้จะทำอะไร ให้ถาม: "Hey, this is my day. How can you help with this?"
- กลับมาทีหลังดูว่าทำอะไรไปบ้าง
- ทำแบบนี้ทั้งสัปดาห์ จะเก่งขึ้นเยอะ
